HTML

(Hyper Text Markup Language) is the most basic building block of the web. It defines the meaning and structure of web content .

CSS

Cascading style sheet (CSS)is a simple mechanisam for adding style to web documents. These pages contain information on how to learn and use CSS and on available software.

Javascript

Javascript is a programming languge.It was originally developed by netscape as a means to add dynamic and interactive elements to websites.

JQuery

jQuery is a lightweight,"write less,do more",Javascript library.The purpose of jQuery is to make it much easier to use Javascript on your website.

Bootstrap

Bootstrap is a framework to help you design websites faster and easier. It includes HTML and CSS based design templates. It also gives you support for javascript plugins.
